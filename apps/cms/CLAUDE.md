# قوانین CMS (apps/cms)

> قوانین ریشه (`/CLAUDE.md`) را هم رعایت کن.

## مسئولیت
مدیریت محتوا: دوره‌ها، اساتید، بلاگ، صفحات، مدیا. پنل ادمین + API محتوا.
حساب کاربری/پرداخت/LMS اینجا نیست (مال apps/api).

## قواعد
- ابزار (Payload/Strapi) در جلسه‌ی قدم ۰ قطعی می‌شود — تا قبل از تصمیم، scaffold نکن.
- مدل هر کالکشن باید با تایپ متناظر در `packages/shared` بخواند (Course, InstructorDetail, ...).
- هر فیلد متنی چندزبانه (عربی + اردو).
- نام فیلدها دقیقاً مطابق قرارداد `docs/API-CONTRACT.md`.
