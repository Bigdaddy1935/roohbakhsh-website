# apps/cms — CMS هدلس

این اپ هنوز scaffold نشده چون انتخاب ابزارش باید در **جلسه‌ی قدم ۰** قطعی شود.

## گزینه‌ها
- **Payload CMS** (پیشنهاد) — با Next.js و TypeScript نیتیو است، RTL و چندزبانه (عربی/اردو) دارد، می‌تواند داخل همین مونوریپو بنشیند. پنل ادمین آماده.
- **Strapi** — بالغ و پرکاربرد، اکوسیستم بزرگ، ولی جدا از Next.

## مسئولیت این اپ (هر چه انتخاب شود)
مدیریت **محتوا**: دوره‌ها، اساتید، بلاگ، صفحات، مدیا. پنل ادمین برای تیم محتوا. ارائه‌ی API محتوا به سایت.
**حساب کاربری / پرداخت / LMS کار این اپ نیست — آن مال apps/api است.**

## بعد از تصمیم
کالکشن‌ها باید با تایپ‌های `packages/shared` هماهنگ باشند (Course, InstructorDetail, ...).
هر فیلد متنی باید چندزبانه (عربی/اردو) تعریف شود.
